Placeholders and Classes Font size 1 placeholderLink

Since: 2.0.0 New Issue

A placeholder to mimic h1 styles. May also be used as a class if $burf-extras is enabled.

Example:

Make a <h4> look like an <h1>.
			.comparison-title {
					@extend %font-size-1;
				}
