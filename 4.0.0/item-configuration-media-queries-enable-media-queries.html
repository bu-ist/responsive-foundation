Configuration » Media Queries Enable media queriesLink

Since: 0.1.0 New Issue

Choose whether or not to enable media queries in your stylesheet. If you set this to false, it will print media queries in the order they are declared so that desktop styles will always override mobile styles, regardless of screen size. This is mostly useful for browsers which do not support media queries. By default, an older IE stylesheet is generated using this variable.
